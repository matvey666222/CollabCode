📘 ИНСТРУКЦИЯ ПО ЗАПУСКУ ПРОЕКТА COLLABCODE
============================================

Данный архив содержит:
- Исходный код веб-приложения CollabCode
- Дамп базы данных MongoDB (коллекции rooms и users)


🛠️ ТРЕБОВАНИЯ ДЛЯ ЗАПУСКА
----------------------------
1. Node.js (версия 16 или выше)
   Скачать: https://nodejs.org/

2. MongoDB Community Edition
   Скачать: https://www.mongodb.com/try/download/community

3. MongoDB Database Tools (для восстановления дампа)
   Скачать: https://www.mongodb.com/try/download/database-tools


💾 ШАГ 1. ВОССТАНОВЛЕНИЕ БАЗЫ ДАННЫХ ИЗ ДАМПА
-----------------------------------------------
1. Распакуйте архив с проектом в любую папку.

2. Откройте терминал (командную строку) и перейдите в папку с дампом:
   cd путь_к_папке/backup

3. Выполните команду для восстановления базы данных:
   mongorestore --db=collabcode ./collabcd

4. Если команда выполнилась успешно, вы увидите сообщения:
   "done restoring collabcode.rooms (32 documents)"
   "done restoring collabcode.users (18 documents)"

5. Проверьте, что база данных создана:
   - Откройте MongoDB Compass (если установлен)
   - Подключитесь к localhost:27017
   - В списке баз данных должна появиться база collabcode


⚙️ ШАГ 2. НАСТРОЙКА ПРОЕКТА
-----------------------------
1. Перейдите в папку server:
   cd Collabcode/server

2. Создайте файл .env (если его нет) и добавьте:
   PORT=3000
   MONGODB_URI=mongodb://localhost:27017/collabcode
   JWT_SECRET=collabcode_secret_key_2025

   (ВНИМАНИЕ: в файле .env НЕ ДОЛЖНО быть строки с облачным подключением mongodb+srv://)

3. Установите зависимости:
   npm install

4. Запустите сервер:
   node server.js

   Если всё правильно, вы увидите:
   ✅ Connected to MongoDB
   🚀 Server running on port 3000


🌐 ШАГ 3. ОТКРЫТИЕ ПРИЛОЖЕНИЯ
-------------------------------
1. Откройте браузер и перейдите по адресу:
   http://localhost:3000

2. Приложение CollabCode запустится.
   - Вы можете зарегистрироваться или войти
   - Создать комнату или присоединиться к существующей
   - Писать код в реальном времени с другими пользователями


📦 СТРУКТУРА БАЗЫ ДАННЫХ (для справки)
---------------------------------------
База данных collabcode содержит две коллекции:

1. users — пользователи
   - name (имя)
   - email (уникальный)
   - password (хешированный)
   - color (цвет аватара)
   - createdAt (дата регистрации)

2. rooms — комнаты для совместного редактирования
   - roomId (уникальный ID)
   - code.html (HTML-код)
   - code.css (CSS-код)
   - code.js (JavaScript-код)
   - createdBy (ссылка на создателя)
   - users (список участников)
   - createdAt (дата создания)
   - lastActive (дата последней активности)


🆘 ВОЗМОЖНЫЕ ПРОБЛЕМЫ И РЕШЕНИЯ
---------------------------------
1. Ошибка "mongorestore не является внутренней командой"
   → Установите MongoDB Database Tools
   → Или выполните команду из папки bin:
     C:\путь_к_tools\bin\mongorestore --db=collabcode ./collabcd

2. Ошибка "ECONNREFUSED" при запуске сервера
   → Убедитесь, что MongoDB запущена:
     - В Windows: откройте "Службы" и запустите MongoDB
     - В Linux: sudo systemctl start mongod

3. Ошибка "Cannot find module"
   → Выполните npm install в папке server

4. Приложение не открывается на порту 3000
   → Возможно, порт занят. Измените PORT в файле .env на другой (например, 3001)
