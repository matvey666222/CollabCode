# CollabCode — Совместная онлайн-песочница для веб-разработки

Веб-приложение для совместного редактирования HTML, CSS и JavaScript кода в реальном времени с чатом и превью результата.

## 🚀 Возможности

- ✍️ **Monaco Editor** — редактор кода как в VS Code
- 👥 **Совместное редактирование** — изменения видны всем участникам мгновенно
- 💬 **Встроенный чат** — общайтесь с командой не покидая редактор
- 👁️ **Превью результата** — сразу видите, как выглядит ваш код
- 📱 **Эмуляция мобильных устройств** — проверяйте адаптивность
- 🎨 **Шаблоны проектов** — быстрый старт с готовыми примерами
- 🔗 **Приглашение по ссылке** — просто поделитесь ID комнаты
- ☁️ **Сохранение в облаке** — код автоматически сохраняется в MongoDB
- 📦 **Экспорт проекта** — скачайте ZIP-архив с файлами
- 🌓 **Тёмная/светлая тема** — настройте редактор под себя
- 🔤 **Настройка размера шрифта** — комфортная работа с кодом

## 🛠️ Технологии

- **Frontend:** HTML5, CSS3, JavaScript, Monaco Editor
- **Backend:** Node.js, Express, WebSocket (ws)
- **База данных:** MongoDB (Mongoose)
- **Аутентификация:** JWT + bcrypt

## ⚙️ Установка и запуск

1. Установите [Node.js](https://nodejs.org/)
2. Настройте MongoDB (локально или [MongoDB Atlas](https://www.mongodb.com/atlas))
3. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/your-username/collabcode.git
   cd collabcode/server


Создайте файл .env в папке server:

PORT=3000
MONGODB_URI=ваша_строка_подключения
JWT_SECRET=ваш_секретный_ключ

Установите зависимости:
npm install

Запустите сервер:
npm run dev

Откройте браузер: http://localhost:3000
